# Kanban Board

Веб-приложение для управления задачами в формате Kanban-доски, построенное на Vue.js 2.

## Описание

Приложение позволяет создавать, редактировать и отслеживать задачи через четыре колонки рабочего процесса:
- **Запланированные задачи** - новые задачи
- **Задачи в работе** - задачи в процессе выполнения
- **Тестирование** - задачи на проверке
- **Выполненные задачи** - завершенные задачи

## Функциональность

- ✅ Создание задач с названием, описанием и дедлайном
- ✅ Редактирование задач
- ✅ Перемещение задач между колонками
- ✅ Удаление задач (только из колонки "Запланированные задачи")
- ✅ Возврат задач из тестирования с указанием причины
- ✅ Отслеживание времени выполнения (во время / просрочено)
- ✅ История редактирования задач
- ✅ Отслеживание причин возврата задач

## Технологии

- Vue.js 2
- Bootstrap 5
- HTML5 / CSS3
- JavaScript (ES6+)

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd Canban_board
```

2. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
# С помощью Python
python -m http.server 8000

# Или с помощью Node.js (http-server)
npx http-server
```

3. Откройте в браузере: `http://localhost:8000`

## Использование

1. **Создание задачи**: Заполните форму вверху страницы (название обязательно) и нажмите "Добавить"
2. **Перемещение задачи**: Используйте кнопку "Далее" для перемещения задачи в следующую колонку
3. **Редактирование**: Нажмите "Редактировать" на задаче, внесите изменения и подтвердите
4. **Возврат задачи**: В колонке "Тестирование" нажмите "Вернуть", укажите причину и подтвердите
5. **Удаление**: Удалить можно только задачи из колонки "Запланированные задачи"

## Особенности

- Автоматическое отслеживание даты создания и последнего редактирования
- Проверка соблюдения дедлайна при завершении задачи
- История причин возврата задач из тестирования
- Адаптивный дизайн с использованием Bootstrap
